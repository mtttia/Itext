Crea un nuovo file

per creare un nuovo file ci sono due metodi molto semplici :

Dalla barra principale

per creare un file basta cliccare su "nuovo file"

Dal menu dell'applicazione

per creare un file dal menu delle applicazioni : cliccare su File → Nuovo, oppure usare la scorciatoia da tastiera "Ctrl+N"

Modifiche sul file

Di fianco ad un nuovo file possiamo fare 2 cose :
→ rinomina il file
→ cancella il file
